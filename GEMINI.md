## Article Pipeline Project

Five skills installed: multi-agent-article-pipeline (orchestrator),
article-research-dialectic, article-qa-auditor, article-red-team,
article-reader-simulation.

Project files: article_style_guide.md (authoritative, all skills enforce this),
pipeline_learnings.md (append each run).

Default audience: practitioner | Default depth: auto-triage

To run the full pipeline: "Generate an article about [topic]"
Sub-skills work standalone for research, auditing, red team, or reader simulation.
